Adds features and functionality to Zoninator

== Description ==

Adds a few helpful features to Zoninator:

* Admin bar button to "Edit Zone"
* Improved date filtering
* Post type filtering
* Improved category filtering
* Adds post thumbnail and post type to the UI
* Automatic 1:1 binding of zones to terms
